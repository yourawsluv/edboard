# edboard
EdTech App

Платформа EdTech 
Product: 
приложение LMS платформы для репетиторов и учеников, на платформе Telegram Mini Apps

Value: 
удобное рабочее пространство и интерактивное обучение
Конкурентный анализ: [Фоксфорд](https://foxford.ru/), [SOHO](https://soholms.com/), [Яндекс](https://practicum.yandex.ru/), ВК доска, Miro, Calendly, VK

CJM:
1. Расписание Shedule
    1. Получение нового слота в Calendly
    2. Просмотр
    3. Переход в занятие
2. Карточка занятия Event
    1. CRUD
    2. Просмотр содержимого
    3. Управление событием
3. Карточка ученика Client 
    1. CRUD
    2. Просмотр данных ученика
    3. Просмотр цепочки событий
    4. Просмотр истории платежей
    5. Просмотр
4. Запуск авто задания
    1. Напоминание об оплате
5. Дашборд
    1. Расписание на день/неделю/месяц
    2. Ближайшее занятие
    3. Оплаты
    4. Ученики
6. Навигация по разделам

JS:
- [ ] Найти ученика - медиа (канал с мемами), биржи, лендинг
- [ ] Активация ученика - Получить данные о балле ОГЭ/ЕГЭ для прохождения
Хранить информацию об ученике и прогрессе
Иметь расписание занятий
Проведение урока в онлайн доске
Иметь информацию о занятии чтобы формировать дз 
Общаться с учеником в мессенджере 

Продуктовые гипотезы:
Рост эффективности - репетитор может снизить времени на подготовку к уроку
Рост качества сервиса для ученика - снижение когнитивной нагрузки
Для репетитора сервис по одному нажатию кнопки, для ученика сервис живого общения с репетитором и интерактивные тренажеры

Архитектура:

Сервис агента - воркер приложения
Сущность - клиентская логика в UI, запросы в API сервиса CRUD на сервере приложения

1. Агент сервис БД Supabase
2. Агент воркер: API
3. Агент сервис Расписание 
4. Агент сервис Автоматизация: авто отправка в мессенджер, 
5. Агент сервис сбор аналитики по занятию
6. Агент сервис учет и визуализация: количество учеников, количество занятий, доход
7. Сущность Карточка занятия: ссылка на звонок и материал
8. Сущность Карточка ученика: данные и прогресс
9. Сущность Интерактивная Доска занятия: рисование, стикеры, вставка изображений, запуск мини приложения 

￼

Стек: 
Python микросервисы, Typescript, shadcn UI
Интеграции: Supabase, Яндекс Телемост, Calendly
